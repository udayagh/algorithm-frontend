Exercise 1 – Palindromes

Create a function which will return true only if a given string input is a palindrome. Your function should return true if the provided string is a palindrome, and false otherwise. A palindrome is defined as a word, verse, or sentence or a number that reads the same backward or forward. For instance the following are examples of palindromes:- • noon • Madam • Able was I ere I saw Elba • 1881 • A man, a plan, a canal, Panama! Please provide suitable test cases to demonstrate that your function works.
